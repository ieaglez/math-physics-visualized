中文

「数理可视课堂」（网站及 iOS 应用）是一个免费的教育工具。我们不收集、不存储、不传输任何个人信息。

  • 应用不需要注册或登录，不请求任何系统权限；
  • 学习进度（"已掌握"标记）和语言偏好仅保存在你自己设备的本地存储（localStorage）中，不会上传到任何服务器；
  • 应用内不包含广告、分析统计（Analytics）或任何第三方 SDK；
  • iOS 应用的全部课程内容打包在应用内，可完全离线使用；
  • 网页版由 GitHub Pages 托管，其服务器可能记录标准的访问日志（如 IP 地址），详见 GitHub 隐私声明。

English

"Math & Physics Visualized" (the website and the iOS app) is a free educational tool. We do not collect, store, or transmit any personal information.

  • No account, no sign-in, and no system permissions are requested;
  • Learning progress ("mastered" marks) and language preference are stored only in your device's local storage (localStorage) and never uploaded anywhere;
  • The app contains no ads, no analytics, and no third-party SDKs;
  • All course content in the iOS app is bundled inside the app and works fully offline;
  • The web version is hosted on GitHub Pages, whose servers may keep standard access logs (e.g. IP addresses) — see the GitHub Privacy Statement.
